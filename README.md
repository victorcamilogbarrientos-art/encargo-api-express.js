# encargo-api-express.js
API RESTful - Sistema de Control de Medicamentos

Este proyecto consiste en una **API RESTful básica** desarrollada con **Node.js** y el framework **Express.js**. Su objetivo es gestionar un conjunto de datos lógicos orientados al control y toma de medicamentos mediante operaciones CRUD completas, devolviendo todas las respuestas en formato **JSON**.

---
Requisitos e Instalación

1. Descarga o clona este repositorio en tu computadora.
2. Abre una terminal dentro de la carpeta del proyecto (`primera_api_js`).
3. Instala Express ejecuntando el comando:
   ```bash
   npm install
