# encargo-api-express.js
# API RESTful - Sistema de Control de Medicamentos

Este proyecto consiste en una **API RESTful básica** desarrollada con **Node.js** y el framework **Express.js**. Su objetivo es gestionar un conjunto de datos lógicos orientados al control y toma de medicamentos mediante operaciones CRUD completas, devolviendo todas las respuestas estrictamente en formato **JSON**.

El proyecto cumple con los requisitos técnicos de separar la lógica de la aplicación en archivos independientes (`index.js` para el servidor, `routes.js` para los endpoints y `data.js` para la persistencia local en memoria).
